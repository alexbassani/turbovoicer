# Repositórios Oficiais e da Comunidade RVC-GUI

O RVC (Retrieval-based Voice Conversion) é uma tecnologia de conversão de voz que ganhou grande popularidade. Embora exista um projeto oficial, a comunidade desenvolveu forks que oferecem interfaces mais amigáveis e funcionalidades aprimoradas. Abaixo, uma lista dos repositórios mais relevantes:

## 1. Repositórios Oficiais do RVC-Project

O projeto original do RVC é mantido sob a organização `RVC-Project` no GitHub. Estes são os repositórios base:

*   **Retrieval-based-Voice-Conversion-WebUI** [1]
    *   **Link**: `https://github.com/RVC-Project/Retrieval-based-Voice-Conversion-WebUI`
    *   **Descrição**: Este é o repositório principal do RVC-GUI, oferecendo uma interface web para treinamento e inferência de modelos de conversão de voz. É a base para muitos dos forks da comunidade.

*   **Retrieval-based-Voice-Conversion** [2]
    *   **Link**: `https://github.com/RVC-Project/Retrieval-based-Voice-Conversion`
    *   **Descrição**: Uma versão mais focada na biblioteca e API do RVC, ideal para desenvolvedores que desejam integrar a funcionalidade de conversão de voz em seus próprios projetos sem a interface web completa.

## 2. Forks e Versões Populares da Comunidade

Diversos desenvolvedores e grupos da comunidade aprimoraram o RVC-GUI, adicionando funcionalidades, melhorando a usabilidade e otimizando o desempenho. Os mais notáveis incluem:

*   **Applio** [3]
    *   **Link**: `https://github.com/IAHispano/Applio`
    *   **Descrição**: Considerado um dos forks mais amigáveis e completos do RVC. O Applio foca na facilidade de uso, oferecendo uma interface intuitiva e suporte a diversas funcionalidades avançadas, incluindo treinamento e inferência. É uma excelente opção para usuários que buscam uma experiência mais refinada que o RVC-GUI original.

*   **W-Okada Voice Changer** [4]
    *   **Link**: `https://github.com/w-okada/voice-changer`
    *   **Descrição**: Este projeto é focado em conversão de voz em tempo real e se destaca pelo seu amplo suporte a hardware, incluindo Nvidia (CUDA), AMD (DirectML/ROCm) e Intel (DirectML/OpenVINO), além de otimizações para Apple Silicon. É uma solução robusta para aplicações que exigem baixa latência.

*   **Mangio-RVC-Fork** (Base para muitos outros forks)
    *   **Descrição**: Embora não seja um repositório ativo como os anteriores, o `Mangio-RVC-Fork` foi uma base crucial para muitos dos desenvolvimentos da comunidade, incluindo o Applio. Muitas das melhorias de usabilidade e performance vistas em forks mais recentes tiveram origem neste projeto. É importante reconhecer sua influência, mesmo que os usuários sejam direcionados a forks mais atualizados.
